# GW2 ChatLinks

PHP encoder/decoder for Guild Wars2 Chat Links.

## Requirements

- PHP 8.1

## Install

```
composer install sugatasei/gw2chatlink
```

## Use

Read the "examples" folder.

## Features

Encode and decode the following chat link types :

- item
- map
- skill
- trait
- recipe
- skin
- outfit
- objective
- build
